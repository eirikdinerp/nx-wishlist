  • Insert list ID to get access to the list that was shared with you
  • Log in to get access to your own and lists shared with you

Registered Users:

  1. Create list
  2. Type of event
  3. Add wishes
  4. Go 'live' with the list and get a list key you can share with your friends.
  5. Alternatively get a viewable link, everyone with the link can see the list
  6. Disable list - archive it

Anonymous Users:

  1. Enter your list key to get access to the list you want to see
  2. Look at list, mark as bought if you want
